# مشروع Gestion

مشروع **Gestion** هو نظام لإدارة وتتبع العمليات الإدارية أو المالية أو الموارد في مؤسسة صغيرة أو متوسطة. الهدف من النظام هو تسهيل تتبع العمليات وتنظيمها، مع إمكانية التوسع مستقبلًا.

## مميزات المشروع
- إدارة بيانات العملاء والموردين.
- إنشاء وتتبع الفواتير.
- تقارير حول الأداء والمعاملات.
- نظام تصاريح بسيط للمستخدمين.

## بنية الملفات

```
/
├── src/              # كود المصدر الأساسي للمشروع
│   ├── components/   # المكونات (Components) القابلة لإعادة الاستخدام
│   ├── pages/        # صفحات التطبيق
│   ├── services/     # الخدمات (API, معالجة البيانات)
│   ├── utils/        # الأدوات والمساعدات البرمجية
│   └── main.*        # نقطة الدخول الأساسية للتطبيق
├── public/           # ملفات عامة (صور، favicon،...)
├── package.json      # تعريف تبعيات المشروع وأوامر التشغيل
├── README.md         # هذا الملف - شرح وتوثيق المشروع
└── ...
```

## كيف تبدأ؟
1. قم بنسخ المستودع:
   ```bash
   git clone https://github.com/bacmat49-stack/gestion.git
   ```
2. انتقل للمجلد:
   ```bash
   cd gestion
   ```
3. ثبّت التبـعيات:
   ```bash
   npm install
   ```
4. شغّل المشروع:
   ```bash
   npm start
   ```
